MoodPlay: an AI-Powered Mood-Based Music Playlist Generator

Skills: Python, Machine Learning (Scikit-learn), Data Analysis (Pandas), Web Development (Streamlit)

Description:

  • Developed an AI-powered web app that uses machine learning to generate music playlists based on user’s mood.
  • Built a Random Forest Classifier to predict moods such as Happy, Sad, Chill, Energetic, and Romantic using audio features (e.g., danceability, energy, acousticness) from a Kaggle Spotify dataset.
  • Achieved 99.25% cross-validated accuracy, ensuring robust mood classification
  • The model performed extremely well on unseen data, with a test set accuracy of 99.35%, ensuring no overfitting and strong generalization to new data.
